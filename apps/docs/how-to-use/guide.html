Simple Design QA - Quick How-To

Capture UX feedback and export issue logs as ZIP

Workflow

  1. Open your app screen and click the Simple Design QA extension icon.
  2. Confirm auto-filled context: Page Name, URL, and Timestamp.
  3. Select Severity and Feedback Type.
  4. Enter a clear issue description in Feedback.
  5. Optional: click Pick Area and select the on-page container where the issue appears.
  6. Optional: add a Suggested Reference Link (plain domains are accepted).
  7. Click Save for each issue.
  8. Use View Logs for filter/search/delete and Export SDQA ZIP at end of QA.

Tips

Use concise, reproducible feedback and keep one issue per log entry. Mark only release-blocking issues as Blocker. Export creates an SDQA_... ZIP with HTML, CSV, and screenshot files for easy sharing.
