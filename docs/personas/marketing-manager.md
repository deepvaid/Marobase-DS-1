# Persona: Marketing Manager

## Profile
- **Name:** James Rivera
- **Role:** Head of Marketing at a growing e-commerce brand (fashion)
- **Age:** 32
- **Tech comfort:** High — uses multiple SaaS tools daily, understands data
- **Team size:** 3 marketers + 1 designer
- **Contact list:** 85,000 subscribers
- **Time in Maropost:** 3–5 hours/day

## Daily Tasks in Maropost (by frequency)

1. **Check campaign reports** — open rates, click rates, revenue attribution (every morning)
2. **Create/schedule campaigns** — write emails, select audience, schedule sends (2–3x/week)
3. **Manage journeys** — monitor automated flows, toggle active/inactive, check conversion (weekly)
4. **Segment audience** — create segments for targeted campaigns (weekly)
5. **A/B test subject lines** — set up split tests, review winners (bi-weekly)
6. **Review acquisition forms** — check signup form performance, conversion rates (monthly)
7. **Build landing pages** — create campaign-specific landing pages (monthly)
8. **Manage content** — upload images, create email templates, dynamic content blocks (ongoing)

## Key Pages Used

| Page | Frequency | What he does |
|------|-----------|--------------|
| Campaign Reports | Daily | Check performance of recent sends |
| Email Campaigns | 3x/week | Create, edit, schedule campaigns |
| Journeys | Weekly | Monitor automated flows, toggle on/off |
| Segments | Weekly | Create audience segments for targeting |
| All Contacts | Weekly | Check subscriber growth, clean lists |
| Acquisition Forms | Monthly | Review form conversion rates |
| Landing Pages | Monthly | Build campaign landing pages |
| Journey Builder | Bi-weekly | Design new automation flows |

## Pain Points

1. **Campaign creation is too many steps** — wants a streamlined wizard, not 7 screens to send an email
2. **No quick preview** — has to save, exit, preview separately. Wants inline preview in the editor
3. **Journey reports are shallow** — can see journey-level stats but not step-by-step conversion funnels
4. **Segment builder is confusing** — AND/OR logic is hard to visualize for complex segments
5. **No A/B test insights** — just shows which version won, not why (no engagement pattern analysis)
6. **Analytics date range is clunky** — wants preset ranges (Last 7d, 30d, 90d) not manual date pickers
7. **Can't duplicate journeys easily** — wants to clone a working journey and modify it

## Success Metrics
- Email open rate > 22% (industry benchmark)
- Campaign revenue attribution growing month-over-month
- List growth rate > 5% per month
- Journey automation covering 60%+ of lifecycle touchpoints

## Interface Expectations
- Visual campaign calendar showing scheduled sends
- Inline email preview (mobile + desktop toggle)
- Drag-and-drop journey builder with clear node connections
- Smart segment suggestions based on engagement data
- Comparison charts: this campaign vs. average, vs. best performer
- Quick-duplicate buttons on campaigns and journeys

## Quote
> "I need to know two things: what's working, and how to do more of it. Don't make me dig through 5 reports to figure out if my last campaign was good."
